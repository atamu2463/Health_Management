# Health_Management

## 概要
従業員自身が心身の体調を記録し、職場メンバーや上長・管理職へ共有できるアプリ。	

## 解決したいこと
### 従業員
- 体調を可視化しセルフケアすることで、安定して長く働けるようになる。
- 自分の特性を理解して、必要なときに必要なサポートを依頼できるようにする。
### 管理者
- 部下やメンバーの体調を把握し、休職者を出さないための予防策を検討しやすくする。結果、心身ともに働きやすい職場環境づくりへ貢献する。
